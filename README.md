# 📊 Projeto Big Data — State of Data 2022

Análise exploratória do dataset **State of Data 2022**, a maior pesquisa sobre o mercado de dados no Brasil, com o objetivo de extrair insights sobre perfis profissionais, salários, ferramentas e tendências da área.

## 🎯 Objetivo
Explorar e tratar um dataset real de grande volume, identificando padrões relevantes sobre a comunidade de dados brasileira.

## 🛠️ Stack
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook / Google Colab

## 📁 Estrutura
- `Projeto_Big_Data.ipynb` — notebook com tratamento e análise dos dados
- `State_of_data_2022.csv` — base de dados utilizada

## ▶️ Como executar
1. Abra o notebook no Google Colab ou Jupyter.
2. Garanta que o arquivo `State_of_data_2022.csv` esteja no mesmo diretório.
3. Execute as células em ordem.
